## 实验环境
- macOS 10.13.6 
- go version go1.9.3 darwin/amd6
- goland 2018.2.1
- truechain-fpow分支

## 代码下载
    git clone -b fpow ow https://github.com/truechain/truechain-engineering-code.git
    
## GoLand下载

    根据自身操作系统选择对应的GoLand版本，可以选择30天试用或者购买正版授权
    
    https://www.jetbrains.com/go/download
